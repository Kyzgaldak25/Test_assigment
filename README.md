# Test_assigment
В данном блоке представлены ответы на квалификационное тестовое задание по продуктовой аналитике и статистике.

**Ключевые навыки, продемонстрированные в работе:**

**Продуктовые метрики:** Расчет MAU, DAU, Retention 1d, Conversion Rate.

**Лояльность:** Расчет индекса NPS.

**Статистика и А/Б тесты:** Интерпретация p-value, анализ статистической значимости, расчет ARPU.

**Визуализация данных:** Понимание типов распределений (бимодальное), дисперсии и корреляции.

**Работа с гипотезами:** Оценка результатов экспериментов и формирование бизнес-рекомендаций.
